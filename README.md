Debezium Embedded Demo
===============

code example for https://dkublik.github.io/2019/06/27/kafka-connectors-without-kafka.html


## Requirements

* Java 11

* Gradle 5.4.1

* Docker 

## To run
import to your favorite IDE and lunch DebeziumCDCDemo.groovy  
or  
from command line:
```
./gradlew clean build
```
